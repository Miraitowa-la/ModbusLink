# ModbusLink v1.3.3 版本发布说明

## 新功能：高级数据类型API扩展

### 异步客户端功能对齐

现在异步客户端（AsyncModbusClient）与同步客户端（ModbusClient）在高级数据类型API方面完全对齐。为异步客户端添加了缺失的高级数据类型读写函数：

### 新增的异步读取函数
```Python
# 异步读取32位无符号整数（占用2个连续寄存器）
async def read_uint32(......)

# 异步读取64位有符号整数（占用4个连续寄存器）
async def read_int64(......)

# 异步读取64位无符号整数（占用4个连续寄存器）
async def read_uint64(......)

# 异步读取字符串（从连续寄存器中）
async def read_string(......)
```

### 新增的异步写入函数
```Python
# 异步写入32位无符号整数（占用2个连续寄存器）
async def write_uint32(......)
    
# 异步写入64位有符号整数（占用4个连续寄存器）
async def write_int64(......)

# 异步写入64位无符号整数（占用4个连续寄存器）
async def write_uint64(......)

# 异步写入字符串（到连续寄存器中）
async def write_string(......)
```

### 文件修改内容

- src\modbuslink\client\async_client.py：新增8个高级数据类型API函数
- 对应示例文件修改：
    - examples\cn - 中文示例
    - examples\en - 英文示例

## 解决议题（ #3 ）

- 修复了异步客户端与同步客户端在高级数据类型API方面的功能差异
- 确保两个客户端类提供一致的用户体验
