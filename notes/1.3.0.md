# ModbusLink v1.3.0 版本发布说明

## 新功能：支持 RS485 模式
在 `RtuTransport` 和 `AsyncRtuTransport` 中添加了原生 RS485 模式支持，通过 RTS 信号实现软件控制的 RS485 总线方向。

``` Python
# Simple usage
transport = RtuTransport('/dev/ttyUSB0', rs485_mode=True)

# Custom settings
from modbuslink import RS485Settings
rs485_settings = RS485Settings(
    rts_level_for_tx=True,
    rts_level_for_rx=False,
    delay_before_tx=0.001,
    delay_before_rx=0.001,
)
transport = RtuTransport('/dev/ttyUSB0', rs485_mode=rs485_settings)
```

## API 新增内容

1. RtuTransport (transport/rtu.py)：
    - 新增了 `rs485_mode: Optional[Union[bool, RS485Settings]]` 参数

2. 异步RTU 传输器（位于“transport/async_rtu.py”文件中）：
    - 新增了 `rs485_mode: Optional[Union[bool, RS485Settings]]` 参数

3. init.py：
    - 增加了 RS485Settings [的重新导出功能，以方便使用
