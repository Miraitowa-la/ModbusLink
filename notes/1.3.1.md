# ModbusLink v1.3.1 版本发布说明

## 新功能：支持日志语言切换设置
在 `ModbusLogger.setup_logging()` 函数中添加了语言配置支持，新的语言参数接受 Language.EN 表示英语或 Language.CN 表示中文（默认）。

``` Python
from modbuslink.utils import ModbusLogger, Language
import logging

# For English logs
ModbusLogger.setup_logging(
    level=logging.DEBUG,
    enable_debug=True,
    log_file='modbus_debug.log',
    language=Language.EN
)

# For Chinese logs (default)
ModbusLogger.setup_logging(
    level=logging.DEBUG,
    enable_debug=True,
    log_file='modbus_debug.log',
    language=Language.CN
)
```

## API 新增内容

1. BilingualLogger (src\modbuslink\utils\logging.py)：
    - 新增了 `BilingualLogger` 类

2. 多个文件中英文适配（位于`src\modbuslink\server` , `src\modbuslink\transport`文件夹中）：
    - 修改了 `async_ascii_server.py` `async_base_server.py` `async_rtu_server.py` `async_tcp_server.py` `data_store.py` `ascii.py` `async_ascii.py` `async_rtu.py` `async_tcp.py` `rtu.py` `tcp.py` 的中英文适配

3. 中英文示例（examples）：
    - 由原有的 `examples` 中英文混合示例，改为 `examples\cn` `examples\en` 中英文示例
